What is Flexbox

The Flexbox Layout (Flexible Box) module is meant to help us lay out elements and modules within websites and applications

Chris Coyier: "The main idea behind the flex layout is to give the container the ability to alter its items' width/height (and order) to best fill the available space... A flex container expands items to fill available free space, or shrinks them to prevent overflow."

Flexbox is helpful for building device-agnostic, responsive web sites and applications.

Getting Started

Flexbox has containers and children.

.flexboxContainer {
		    display:flex;
		    flex-direction: row;
		    flex-wrap: wrap;
		    justify-content: flex-start;
		    align-content: flex-start;
		    align-items: center;
		}
				
		 
		.FlexboxItem {
		    width: 300px;
		    height: 250px;
		    flex-grow: 0;
		    flex-shrink: 0;
		    flex-basis: 0;
		}

Above would be an example of how to setup a flexbox container with a single element inside. The children of a display:flex container will typically be called items to make organization easier. Here is an example of a flexbox contianer:

Choosing Item Direction and Alignment

The above is a flexbox container. The items inside will remain equidistance and of equal size no matter the screen size.flex-direction will change which direction your items are aligned. justify-content will use the free space remaining on the screen either around or between the items. Around adds space on both sides equally, while between pushes the elements away from one another to the max container distance.

justify-content can also be used to align the items on either the horizontal axis while using row or vertical axis when using column. justify-content can also have the items begin at flex-start or flex-end to begin the items at the beginning or end of the parent container.

align-items is used to align the items along the alternate axis compared to justify-content. This is very powerful and can be used to vertically and horizontally align elements right in the center of a container:

Making Items Flexible

Now that we can make the container place the items in the direction we want and align them to the container as we need them we can choose how they will flex within the container. This is done by modifying the elements flex properties. flex-grow is used to define the item size relative to the other flex items that share it's container. flex-shirnk will determine how much faster the item shrinks when the container gets smaller. The higher the shrink the more of the container space will go to the other items. flex-basis will be used to set a minimum width of the item. The item will take up all the space it can in the container and get as small as the flex-basis before it will no longer shrink or be moved to the next row depending on the flex-wrap: property.

The container above is duplicated from the previous example. The align-items property is set to stretch however and height is removed from the item. Each item has the following CSS:

.flexBoxItem {
		    flex-grow: 1;
		    flex-shrink: 0;
		    flex-basis:200px;
		    }

This means that the items will grow at the same pace as one another to fill the available space. Each row/column will fill with the flex-items until the items reach their flex-basis: 200pxAgain, the align-items: stretch; property allows these items to take up as much space that is available.
